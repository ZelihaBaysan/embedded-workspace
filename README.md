# Daryl Dixson Hakkında Soru-Cevap Programı

Bu Python programı, Daryl Dixson hakkında önceden belirlenmiş 5 soruya kullanıcıdan gelen sayı girişiyle cevap veren basit bir komut satırı uygulamasıdır. Kullanıcı, 1 ile 5 arasında bir sayı girerek ilgili sorunun cevabını görebilir ve istediği kadar soru sorabilir. Program `exit` komutuyla sonlandırılabilir.

## Amaç

Bu programın amacı, llm-repo-assistant reposunu test etmek için, Daryl Dixson hakkında temel bilgileri hızlı ve pratik şekilde sunmak ve kullanıcıya interaktif bir soru-cevap deneyimi sağlamaktır.
